E.W. MachinesTech – Trial Website Documentation

1. Project Overview
This project is a trial HTML website created for my father’s business, E.W. MachinesTech, which deals in the import and export of industrial machines. The purpose of this website is to establish a basic online presence, provide essential business information, and showcase the types of machines handled by the company.
The website is developed using pure HTML and internal CSS, making it lightweight, easy to maintain, and suitable for a small-scale business introduction.

3. Objectives of the Project
To create a simple and professional business website
To provide company details and contact information
To list major machines handled by the business
To learn and demonstrate basic web development skills
To serve as a foundation for future expansion (JavaScript, backend, hosting)

4. Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and layout (internal CSS)
Web Browser – To run and test the website locally
